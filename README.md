# Where Am I?

Where Am I is a simple demo iOS application for determining the user's current location and retrieving nearby points of interest.

The points of interest include: 
- Cafes
- Restaurants
- Parks
- Hospitals
- Police
- Fire departments

## Requirements
- iOS 13.0+
- XCode 12.5.1
